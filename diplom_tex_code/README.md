# Бакалаврская ВКР — Юрченко В. А., МФТИ ФПМИ, 2026

Тема: **Оценка опционов, выплата по которым зависит от траектории, в том числе с помощью методов машинного обучения**

## Состав файлов

- `main.tex` — главный файл, собирает всё через `\input`.
- `preamble.tex` — пакеты, шрифты, геометрия, заглушки.
- `refs.bib` — библиография в формате BibLaTeX.
- `sections/`
  - `title.tex` — титульный лист (МФТИ ФПМИ).
  - `abstract.tex` — аннотация RU + EN.
  - `introduction.tex` — введение.
  - `chapter1_exotic.tex` — глава 1, экзотические деривативы.
  - `chapter2_correlation.tex` — глава 2, корреляционные свопы и rainbow.
  - `chapter3_ml.tex` — глава 3, ML.
  - `chapter4_data.tex` — глава 4, данные.
  - `chapter5_results.tex` — глава 5, эксперименты.
  - `conclusion.tex` — заключение.
  - `appendix.tex` — приложения.

## Как компилировать

```bash
xelatex main.tex
biber main
xelatex main.tex
xelatex main.tex
```

Нужен XeLaTeX (или LuaLaTeX): `pdflatex` не подойдёт, потому что шрифты подключаются через `fontspec` (Times New Roman). На Overleaf достаточно выбрать `XeLaTeX` в настройках проекта.

## Что внутри прототипа

- Структура и оформление под требования МФТИ ФПМИ (14 pt Times, полуторный интервал, поля 30/15/20/20 мм).
- Все главы и подразделы, перечисленные в твоём ТЗ.
- Математика из лекции (модели constant, mean-reverting, Якоби, Фишер–OU) переписана из beamer-формата в нормальные нумерованные уравнения с пояснениями.
- Заглушки:
  - `\placeholderfig{ширина}{высота}{подпись}{метка}` — зелёный квадрат под график.
  - `\placeholdertab{подпись}{метка}` — пустая таблица.
  - `\todoblock{...}` — рамка с TODO-пометкой по ходу текста.
